About This Security Demonstration

This demo illustrates how attackers can use the HTML5 Fullscreen API to create convincing fake interfaces that mimic legitimate websites and operating systems.

What You'll See:
• When you click the YouTube link, the page will request fullscreen permission
• Once granted, a fake browser and OS interface will appear
• Any interaction will reveal that you've been "phished"

Remember: Always verify website URLs before entering sensitive information and be cautious of unexpected fullscreen requests.
